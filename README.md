# Bricks Pyramid

```text
[]
[][]
[][][]
[][][][]
```

Build a pyramid, one brick at a time.

This tiny Python script takes your brick count and calculates the maximum possible pyramid height.

## How It Works

```text
Layer 1 -> 1 brick
Layer 2 -> 2 bricks
Layer 3 -> 3 bricks
...
```

The script keeps building full layers until there are not enough bricks left.

## Run It

```bash
python Bricks_Pyramid.py
```

Enter a number of bricks, and get the pyramid height instantly.
